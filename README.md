# covid-bronze-silver-architecture
Arquitectura de datos en capas Bronze–Silver aplicada a registros clínicos nacionales en México (COVID-19, 2020–2025). Incluye scripts SQL y Python para ingesta, transformación, normalización y consolidación de más de 56 millones de registros en PostgreSQL.
